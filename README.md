# Skill Gap Matcher AI

Skill Gap Matcher AI is an AI-powered career guidance application that helps users explore suitable job roles, understand role requirements, and generate personalized learning roadmaps based on their current skills.

---

## Overview

Users manually enter the skills they already have.
Using AI, the application suggests relevant job roles, summarizes job responsibilities, and generates structured learning roadmaps to help users progress in their careers.

---

## Features

* Manual skill input after user login
* AI-generated job role suggestions based on entered skills
* Sidebar with multiple job roles
* AI-powered job role summarization
* Personalized learning roadmap generation for any skill
* Simple and intuitive UI with smooth scrolling and sidebar navigation

---

## User Flow

1. User logs in to the application
2. User enters their existing skills manually
3. User scrolls and clicks the **Generate** button
4. AI suggests relevant job roles
5. User selects a role from the sidebar to view an AI-generated summary
6. User enters a skill in the roadmap section
7. AI generates a step-by-step learning roadmap

---

## Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* AI: Large Language Model APIs
* Authentication: User login system
* UI: Responsive layout with sidebar navigation
